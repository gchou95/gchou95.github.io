Research Projects

Using neural networks for fast reachable set computations [paper]

Collaborators: Frank Jiang, Mo Chen, Claire Tomlin.

Overview: Hamilton-Jacobi reachability is a powerful tool that provides theoretical guarantees on system safety and performance, but its computation scales exponentially with system dimension. There exist some approaches for reachable set approximation that somewhat alleviate the exponential scaling, but often require stringent assumptions on the system. Using a neural network, we formulate a method without such restrictions that scales better in time and space complexity by localizing the reachable set computation.

Contributions:

  • Developed and implemented robust gradient-based method for control synthesis using the neural network output.
  • Participated in development and implementation of a dynamic training strategy and methods for data generation and optimization.
  • Participated in design of neural network topology.

Multi-vehicle collision avoidance via Hamilton-Jacobi reachability

Collaborators: Aparna Dhinakaran, Mo Chen, Prof. Claire Tomlin.

Overview: Prior work using mixed integer programming has resulted in safety guarantees for systems of three UAVs, but this method does not generalize to systems with more UAVs. We propose a hybrid framework that uses Hamilton-Jacobi reachability for formal verification of systems of N+1 vehicles, where N is the number of vehicles that can be safely avoided using an existing collision avoidance algorithm.

Contributions:

  • Participated in design of collision-avoidance algorithm.
  • Implemented numerical simulations of algorithm.

Motion planning via gradient optimization and learning from experience

Collaborators: Prof. Anca Dragan.

Overview: The goal of this project is to improve the performance of the state-of-the-art motion planner TrajOpt. While TrajOpt has high success rates, the types of environments and constraints that tend to trigger failures are unclear. We aim to better characterize the cases that cause TrajOpt to fail and improve performance by learning initializations that will improve convergence to a safe trajectory, even in the presence of humans in the environment. We also aim to learn trajectories from experience to avoid repeated failures.

Contributions:

  • Implementing a random, realistic environment generator to serve as a testbed for characterizing hard problems for TrajOpt.
  • Designing and implementing framework to learn optimizer initializations that improve convergence to a feasible trajectory.
  • Designing and implementing a framework to learn trajectories from experience.

Course Projects

Control for vehicle docking and payload transfer

Course: Mechanical Engineering Laboratory.

This project is inspired by the midair refueling control problem and involves a system of two cars, where one car is teleoperated and another car is autonomously controlled, moving on a bumpy track. The goal is to control the autonomous car to dock and deliver a payload to the teleoperated car. We implemented an overhead vision system and an ultrasonic sensor array to provide position and velocity data for a PID controller.

Optimal path planning for small mobile robots

Course: Introduction to Robotics.

This project is inspired by the need for cheap navigation and mapping in unsafe environments, such as an unstable building after a natural disaster. We use cheap, small, mobile robots (Zumies) to navigate cluttered environments to find a hidden goal state, which can only be detected by the cameras mounted on the Zumies. We implemented an overhead vision system and a Bellman-Ford path planning algorithm to control the Zumies to find the goal efficiently.
